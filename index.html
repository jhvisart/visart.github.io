Qué es esta página

Es una portada principal para reunir todos los proyectos en un solo lugar. Funciona muy bien como página base del usuario en GitHub Pages y da una presentación más seria que solo mostrar repositorios.

Tecnologías y enfoque

HTML, CSS y JavaScript pueden convivir aquí con demos visuales, canvas, SVG, animaciones o WebGL. Esta plantilla está pensada para verse moderna, limpia y visualmente atractiva.
